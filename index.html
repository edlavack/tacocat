TacocaT: A Coding Challenge

TacocaT is a small scale application I built for palindromes. For instance, "go hang a salami I'm a lasagna hog" or "Tacocat" would be considered palindrome as it is spelled the same forwards and backwards!

Simply type in a word or phrase, and the application will let you know whether or not it is a palindrome. If so, it will display the phrase. If not, you'll be asked try again!
